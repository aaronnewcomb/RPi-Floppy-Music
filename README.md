RPi-Floppy-Music
================

This is a utility written in python to covert and play midi files on floppy drives connected to a Raspberry Pi.

Setup
=====

Connect up to 8 floppy drives to the RPi.
(more info here)

Installation
============

```pip install mido```

Usage
=====
